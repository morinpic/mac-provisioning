# macbook-provisioning

## Setup
- Install application
```
$ ./setup.sh
```

- Manual install
  - Local by Flywheel  
https://localbyflywheel.com/

- Syncing configration file
  - iTerm2
  ```
  ~/Dropbox/Mackup/Library/Preferences/com.googlecode.iterm2.plist
  ```
  - Alfred
  ```
  ~/Dropbox/Mackup/Alfred.alfredpreferences
  ```
- Reboot to mac
